# Browthon-Reborn

Latest version of Browthon. A clean version in Python with PyQt5

## Dependances

- Python 3.5+
- PyQt5 (pip install PyQt5)

Warning : If you are on Windows, you must have **Python 64Bits**. If you have Python 32Bits, PyQt don't have web engine.
