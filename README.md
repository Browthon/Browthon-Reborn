# Browthon-Reborn

Latest version of Browthon. A clean version in Python with PySide2

## Dependances

- Python 3.5+
- PySide2 (pip install PySide2)

## Install

- Launch in cmd : `pip install Browthon-Reborn`
- Launch in cmd : `browthon`
- Enjoy
